DARIA API Documentation

GET/api/health

Check the health status of the API.

Response

{
    "status": "ok",
    "timestamp": "2025-05-24T23:57:52.497677"
}

GET/api/sessions

Get a list of all interview sessions.

Response

[
    {
        "session_id": "uuid",
        "title": "Interview Title",
        "status": "active",
        "created_at": "timestamp",
        "conversation_history": []
    }
]

POST/api/interview/start

Start a new interview session.

Request Body

{
    "title": "Interview Title"  // optional
}

Response

{
    "success": true,
    "session_id": "uuid",
    "message": "Hello! How can I help you today?"
}

POST/api/interview/{session_id}/message

Add a message to an interview session.

Request Body

{
    "message": "User's message"
}

Response

{
    "success": true,
    "message": "Message added successfully"
}

DELETE/api/interview/{session_id}

Delete an interview session.

Response

{
    "success": true,
    "message": "Interview deleted successfully"
}
